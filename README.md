## Publish with GitHub Pages

1. Create a public GitHub repository, for example `pixel-beader-site`.
2. Push this folder to the repository's `main` branch.
3. In GitHub, open Settings -> Pages.
4. Set Source to `Deploy from a branch`.
5. Set Branch to `main` and folder to `/ (root)`.
6. Save. GitHub will publish the site after the Pages build finishes.

The published URLs will look like:

- Marketing URL: `https://YOUR_GITHUB_USERNAME.github.io/pixel-beader-site/`
- Support URL: `https://YOUR_GITHUB_USERNAME.github.io/pixel-beader-site/support/`
- Privacy Policy URL: `https://YOUR_GITHUB_USERNAME.github.io/pixel-beader-site/privacy/`
