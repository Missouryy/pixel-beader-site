Pixel Beader Support

Need help with a project, export, purchase, or restore flow? Send a message and include your device model, iOS version, and a short description of what happened.

Email Support

Restore purchases

Open Pixel Beader, go to Settings, choose Membership, then tap Restore Purchases. Make sure you are using the same Apple Account that made the purchase.

Membership options are unavailable

Check your network connection and App Store account, then return to Settings and tap Refresh. If the issue continues, contact support.

Export a 3D snapshot

Open a project, move to the View stage, then use Export 3D Snapshot. The image is saved to Photos after you grant photo library permission.

Import photos

When creating a canvas project, choose Select Photo and allow photo library access. Pixel Beader processes images on device.
